# Amazon Clone

A simple Amazon homepage clone built using HTML and CSS.  
This project focuses on creating a clean UI and replicating the layout of the Amazon website.

---

## Features
- Navigation bar similar to Amazon
- Product sections layout
- Clean and modern UI design
- Responsive homepage design

---

## Technologies Used
- HTML
- CSS
